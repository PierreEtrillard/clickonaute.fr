Je conçois des API en JavaScript avec Express ou en Java avec Spring Boot. Spring Web est ma bibliothèque de prédilection, je vous explique pourquoi.

Pourquoi ?

Créer des API avec Spring est la garantie d'exploiter au mieux la robustesse et la performance de Java.

Les classes utilitaires fournies par Spring Web facilitent la production.

Spring jouit d'une documentation fournie et d'une grande communauté d'utilisateurs (60 % des développeurs Java utilisent Spring).

La stabilité du framework a fait ses preuves au fil des années. Aujourd'hui, Spring détient 48 % des parts de marché parmi les frameworks Java d'entreprise.

J'effectue l'installation et la configuration de bases de données MySQL ou MongoDB localement, ou j'établis la connexion de vos API à des services distants comme Atlas.

La base de données selon

Les bases de données Mongo se démarquent des bases de données relationnelles par leur architecture en documents.

Cela offre des gains de performances en évitant par exemple les chargements de données volumineuses, comme dans les cas de requêtes n+1 sur des bases de données SQL.

Asynchrone, Mongo est une base de données idéale dans le contexte d'applications réactives exigeant une disponibilité permanente des données.

J'utilise aussi des bases de données SQL pour des projets nécessitant une cohérence transactionnelle infaillible.

Le couteau suisse de JEE

Le module Spring Data JPA s'appuie sur Hibernate pour préparer des requêtes performantes, standardisées et gérer automatiquement le déroulement des transactions avec les bases de données (SQL ou NoSQL).

Par exemple, en utilisant l'annotation @Projection(), on réduit considérablement le nombre et la taille des transactions avec une base de données SQL.

Facilitez la connexion de vos utilisateurs finaux grâce au standard OIDC

Grâce à Spring Security, je connecte vos API à des services d'authentification tiers (Google, Microsoft, Facebook...) simplifiant l'authentification de vos utilisateurs.

Avec le standard OIDC, il est ensuite facile d'exploiter les données utilisateurs dans le respect du RGPD.
