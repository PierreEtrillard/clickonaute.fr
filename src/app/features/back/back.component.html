Pourquoi ?

Créer des API avec spring est la garantie d'exploiter au mieux la robustesse et la performance de Java

Les class utilitaires fourni par Spring Web faciltent la production

Spring joui d'une documentation fournis et d'une grande communnauté d'utilisateurs (60% des développeurs Java utilisent Spring).

La stabilitée du framework à fait ses preuves et au fil des années, aujourd'hui Spring détient 48% des part de marché parmi les frameworks Java d'entreprise.

J'effectue l'installation et la configuration de base de donnée mySql ou MongoDb localement, ou etablie la connexion de vos api à des services distants comme Atlas.

La base de données selon

Les Bases de donnée Mongo se démarquent des bases de données relationnelles par leurs architectures en documents.

Cela offre des gain de performances evitant par exemple les chargements de données volumineuses comme dans les cas de requètes n+1 sur des bases de données sql.

Asynchrone, Mongo est une base de donnée ideal dans le contexte d'application réactives exigeant une disponibilitée de data permanente.

J'utilise aussi des bases de données sql pour des projets réclamants des cohérences transactionelles infaillibles.

le couteau suisse JEE

Le module Spring Data Jpa s'appuie sur Hibernate pour préparer des requètes performante, standardisées et gérer automatiquement le déroulement des transactions avec les bases de données (sql ou noSql).

Par exemple en utilisant l'annotation @Projection() on reduit considérablement le nombre et la tailles de transactions avec une Base de données SQL

Faciltez la connexion de vos utilisateurs finaux grâce au standart OIDC

Grace à Spring Security, je connecte vos API à des services d'authentifications tiers (google, microsoft, facebook...) simplifiant l'authentification de vos utilisateurs

Avec le standart OIDC il est ensuite facile d'exploiter les données utilisateurs dans le respect de la RGPD.
