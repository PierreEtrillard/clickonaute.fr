Le monde des bases de données

  • Installation et configuration de bases de données MySQL pour des projets nécessitant une cohérence transactionnelle infaillible.
  • Installation et configuration de bases de données MongoDb local ou connexion au service cloud Atlas pour des données disponibles à tout instant.

La base de données selon

Les bases de données Mongo se démarquent des bases de données relationnelles par leur architecture en documents.

Cela offre des gains de performances en évitant par exemple les chargements de données volumineux, comme dans les cas de requêtes n+1 sur des bases de données SQL.

Asynchrone, Mongo est une base de données idéale dans le contexte d'applications réactives exigeant une disponibilité permanente des données.

Le couteau suisse JEE

Le module Spring Data JPA s'appuie sur Hibernate pour préparer des requêtes performantes, standardisées et gérer automatiquement le déroulement des transactions avec les bases de données (SQL ou NoSQL).

Par exemple, en utilisant l'annotation @Projection(), on réduit considérablement le nombre et la taille des transactions avec une base de données SQL.
