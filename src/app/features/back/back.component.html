Pourquoi ?

Créer des API avec spring est la garantie d'expmoiter au mieux la robustesse et la performance de Java

Les class utilitaires fourni par Spring Web faciltent la production

Spring joui d'une documentation fournis et d'une grande communnauté d'utilisateurs (60% des développeurs Java utilisent Spring).

La stabilitée du framework à fait ses preuves et au fil des années, aujourd'hui Spring détient 48% des part de marché parmi les frameworks Java d'entreprise.

La base de données selon

Les Bases de donnée Mongo se démarquent des bases de données relationnelles par leurs architectures en documents.

Cela offre des gain de performances evitant par exemple les chargements de données volumineuses comme dans les cas de requètes n+1 sur des bases de données sql.

Asynchrone, Mongo est une base de donnée ideal dans le contexte d'application réactives exigeant une disponibilitée de data permanente.

J'utilise aussi des bases de données sql pour des projets réclamants des cohérences transactionelles infaillible.
